# OfflineStaticProtection

Torch plugin for Space Engineers that protects player grids while they are offline.

## Features
- Locks static grids when player disconnects
- Disables all production blocks while offline
- Unlock delay with countdown messages
- Fully configurable
- No SafeZone usage
- No Torch commands

## Configuration
Edit `OfflineProtection.cfg` in plugin storage folder.

## Build
Run `build.bat` to generate ZIP ready for Torch.
